# bijlages-HER2
de bijlages voor de borstkanker project

## Ai gebruik
tijden de project werd alleen AI gebruikt voor het oplossen van foutmelding tijdens coderen en korter samenvatten van tekst.

## gebruikte packages
De eiwitten die betrokken waren bij EMT transitie waren bepaald via KEGG-pathway analyse in R met de packages: pathview (V 1.46.0)^5, clusterProfiler (V 4.14.6)^6 en org.Hs.eg.db (V3.20.0)^7. De eiwitten waren met de Benjamini Hochwald (p<0.05)  uit de KEGG-pathways gehaald en in een schematische overzicht gezet. Voor de EMT-score is gewerkt met de packages ggplot2 (V4.0.2)^8, dplyr (V1.2.0) 9, tidyr (V1.3.2)^10. Voor de Go analyses: STRINGdb (2.20.0)^11 igraph (V2.2.2)^12, tidygraph (V1.3.1)^13
